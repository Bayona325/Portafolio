### 🔧 Pasos para desplegar
```bash
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/Bayona325/Portafolio.git
git push -u origin main
```

Luego, en GitHub:
1. Ir a **Settings → Pages**
2. En *Source*, seleccionar `Deploy from a branch`
3. Elegir la rama `main` y carpeta `/ (root)`
4. Guardar cambios ✅

Tu sitio quedará disponible en:
```
https://bayona325.github.io/Portafolio/
```
