Descripción: Plataforma web completa para que desarrolladores y entusiastas de videojuegos compartan, comenten y evaluén conceptos de juegos. Proyecto full-stack con autenticación, base de datos relacional y API RESTful.

Características principales:

  • Sistema de autenticación de usuarios seguro
  • Galería de conceptos con búsqueda y filtros
  • Comentarios y sistema de puntuación (likes)
  • Perfil de usuario personalizable
  • Panel administrativo para gestión de contenidos
  • Interfaz responsiva y moderna

Tecnologías utilizadas:

  • Frontend: JavaScript, React, HTML5, CSS3
  • Backend: Node.js / Python (API REST)
  • Base de datos: MySQL
  • Control de versiones: Git & GitHub
